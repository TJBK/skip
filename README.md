# skipbot
Bot that will auto cell people upon mention

## Requirements
[Node.js](https://nodejs.org/en/) 12.10.0

## Installation
Use NPM to install

```bash
npm install
```

## Usage
First fill out the config.json

```bash
npm run start
```
